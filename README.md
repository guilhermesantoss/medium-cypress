# Testes E2E com Cypress

## Executando o projeto

Instale as dependÃªncias do projeto utilizando o comando:
```bash
// usando NPM
$ npm install

// usando YARN
$ yarn install
```

Deixe o projeto rodando com o seguinte comando:
```bash
// usando NPM
$ npm start

// usando YARN
$ yarn start
```

Agora execute o seguinte comando para ver os testes acontecendo no Browser:
```bash
// usando NPM
$ npm run test:e2e

// usando YARN
$ yarn test:e2e
```
